# Tipos de datos

## Funciones declaradas

Para especificar los tipos de datos en una función declarada de TypeScript, luego de indicar los parámetros, se indica el tipo de dato. Por ejemplo:

```ts
function greet(name: string): string {
  return `Hola ${name}`;
}
```

## Parámetros

Para especificar tipos de datos en los parámetros en las funciones (Esto es algo que TypeScript exige), luego del nombre se indica dos puntos y el tipo de dato (Véase ejemplo anterior.).
