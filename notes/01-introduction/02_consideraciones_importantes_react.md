# Usar React como una librería o un framework

> **Nota:** El curso utilizara React como una librería para facilitar la enseñanza del mismo y enseñar la utilizad de diferentes librerías.

La idea de una librería es permitir al desarrollador decidir qué herramientas externas para usar junto a React. En contraste, la idea de un framework es ofrecer todas las herramientas.

Aunque React es considerado cómo una librería, también se puede usar como un framework. Pero la documentación menciona en resumen estos detalles:

1. Si se desea utilizar como una librería, se recomienda usar **Vite**, **Parcel** o **RSbuild**. Al usar React como una librería, se puede utilizar React con las siguientes herramientas:
   1. **Para la gestión de rutas** está React Router o Tanstack Router.
   2. **Para el Data Fetching (Traer información de fuentes externas)** se recomienda React Query, SWR o RTK Query.
2. Si se desea utilizar como un framework, se recomienda usar **Next.js**, **React Router** o **Expo (Con React Native para aplicaciones nativas)**.

> **Nota:** La ventaja de Vite es que permite tener un ambiente moderno de desarrollo. Además que es un “bundler”.
